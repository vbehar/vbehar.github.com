Helm Ecosystem

Helm is a package manager for Kubernetes. It is the tool most people use to package their applications, and then deploy on Kubernetes. It is also the core of an ecosystem growing every day: plugins, wrapper tools, charts, …

I have been contributing to a few open-source projects in the Helm ecosystem:

  • Helmfile - a “declarative spec for deploying helm charts”
  • Helm Unit Tests plugin - a plugin for running unit tests on Helm charts
  • Helm Tiller plugin - a plugin for running Helm in “tiller-less” mode in Helm 2
  • Helm charts - various contributions to common charts I’ve been using

Project link: https://helm.sh/
