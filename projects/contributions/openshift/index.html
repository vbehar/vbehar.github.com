OpenShift

OpenShift is a Kubernetes “distribution” developed by Red Hat.

In the summer of 2015 I started using this newly announced platform, and as always when working with new projects, found a lot of opportunities for improvements.

This is how I contributed new features, bug fixes, bug reports and documentation in:

  • OpenShift Origin - including support for custom Dockerfile path when building Docker images
  • OpenShift source-to-image - the tool used to build container images without using Dockerfiles
  • OpenShift Documentation

I also contributed to the OpenShift completion for oh-my-zsh plugin.

Project link: https://github.com/openshift/origin
